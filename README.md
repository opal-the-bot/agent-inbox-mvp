# Agent Inbox MVP

A lightweight web app where humans assign tasks to AI agents, track status, resolve blockers, and approve actions.

## MVP Scope
- Task board (Backlog / In Progress / Blocked / Done)
- Blocker queue (questions requiring human input)
- Daily brief panel (summary, approvals needed, next priorities)

## Stack (v1)
- Next.js + TypeScript + Tailwind
- Local JSON persistence for prototype
- Vercel deployment
